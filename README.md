# 食用指南

（本实验为FZU的compiler-lab）

## 编译

在项目根目录下

```sh
make
```

## 运行

在项目根目录下

```sh
./main
```

即可

## 清理

若要重新编译，清除`.o`文件和`main`文件，输入：

```sh
make clean
```
