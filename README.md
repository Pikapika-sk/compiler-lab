# 食用指南

（本实验为FZU的compiler-lab）

## 编译

在项目根目录下

```sh
make
```

## 运行

### 默认方式

默认使用的是/test目录下的`test.lex`文件作为测试文件，在项目根目录下

```sh
./main
```

即可

### 自定义方式

如果要自己定义文件，可以自行创建，在执行时输入路径即可：（假设我在test下新建了文件`new.lex`）

```sh
./main test/new.lex
```

## 清理

若要重新编译，清除`.o`文件和`main`文件，输入：

```sh
make clean
```
